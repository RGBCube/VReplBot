# v-repl-bot

A simple Discord bot that runs your V code.

## Installation

```sh
git clone https://github.com/RGBCube/v-repl-bot
cd v-repl-bot
python3 -m pip install -U poetry
poetry install
```

## Running

```sh
poetry run bot
```
